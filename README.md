# HTML, CSS, JS 기초

## HTML

> HTML : Hyper Text Markup Language - 마크업(표시) 언어 
> 
> 웹페이지의 구조를 표시
> 
> 웹페이지의 콘텐츠를 표현

### HTML Elements

> Element: 요소 - 의미적
> 
> Tag : 태그 - 기술적  

> 문법(Syntax)
> 
> <>로 감싸즘
> 
> 소문자 사용
> 
> 시작태그와 종료태그 세트로 구성됨.
```
<tagname>contents</tagname>
```

> 예외) 시작태그만 존재하는 경우 : 빈 요소(Empty Element)
> 포함 관계(Nested)
```
<body> 
  <div>
    text
  </div>
</body>
```
